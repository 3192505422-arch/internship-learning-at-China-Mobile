# 🧠 TRAE 中创建 Skill

> 让 AI 告别"间歇性失忆"，把经验打造成可复用的知识包。

---

## 一、什么是 Skill？

Skill 解决的核心问题：**让 AI 彻底告别间歇性失忆。**

> 只需要告诉它想做什么，就能自动调取写好的技能包，按照流程瞬间搞定。

---

## 二、Skill 的基本结构

本质上是一个**普通文件夹**，核心结构如下：

```
你的Skill名/
├── 📄 skill.md          ← **唯一必须的文件**，写清技能叫什么、什么时候触发、执行步骤
├── 📁 examples/         ← (可选) 放一些范本 .md
└── 📁 resources/        ← (可选) 放复杂的背景资料和自动化脚本
```

> 💡 **简单来说：** 把经验打造成"知识包"，随手就能用。

---

## 三、如何在 TRAE 中创建 Skill

### 方式一：下载别人的 Skill 包

在 TRAE 设置里点击 **"规则和技能"** → 选择 **"创建"** → 导入文件夹。

### 方式二：对话式创建

在 Solo 模式下，输入指令让 AI 帮你建：

```text
帮我建一个 ... 性能的 Skill
```

### 方式三：手动创建

在标准的 Markdown 模板里填写：

| 字段 | 说明 |
|------|------|
| 🏷️ **技能名字** | 给你的 Skill 取个名字 |
| ⚡ **触发条件** | 什么时候调用这个 Skill |
| 📋 **操作步骤** | 详细的执行流程 |

> 复杂的流程可以慢慢往里面加具体的执行指令。

---

## 四、如何在 TRAE 中使用 Skill

### 显式调用

直接告诉 TRAE 用哪个技能：

```text
用 [技能名] 来帮我做这件事
```

### 隐性调用

在创建 Skill 时，把 **"什么时候用"** 写清楚，TRAE 会在合适的场景下自动调用。
