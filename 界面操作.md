# 🖱️ 界面操作

> 定位到元素之后，如何对元素进行操作。

---

## 一、元素通用操作

### 1. 获取元素内部的可见文本

通过 Locator 对象的 `inner_text()` 方法可以获取元素的内部文本。

如果 Locator 匹配到多个元素，可以使用 `all_inner_texts()` 获取所有匹配的文本（列表形式返回）。

### 2. 获取元素内部的不可见文本

使用 Locator 对象的 `text_content()` 或 `all_text_contents()` 方法获取单个或所有匹配对象的文本（包含隐藏元素）。

```python
# 示例 HTML：
# <p id="source">
#   <span id='text'>看一下<br>这个内容<br>如何变化</span>
#   <span style="display:none">隐藏内容</span>
# </p>

lc = page.locator("#source")
print('innerText:  ', lc.inner_text())    # 只看可见文本
print('--------------')
print('textContent:', lc.text_content())   # 包含隐藏文本
```

### 3. 其他通用操作

| 操作 | 方法 | 说明 |
|------|------|------|
| 🔗 获取元素属性 | `.get_attribute('属性名')` | 获取指定属性值 |
| 📄 获取内部 HTML | `.inner_html()` | 获取元素内部的 HTML 结构 |
| 🖱️ 单击 | `.click()` | 模拟鼠标单击 |
| 🖱️ 双击 | `.dblclick()` | 模拟鼠标双击 |
| ✋ 悬停 | `.hover()` | 模拟鼠标悬停在元素上 |
| ⏳ 等待可见 | `.wait_for()` | 等待元素变为可见 |
| 👁️ 判断可见 | `.is_visible()` | 立即判断元素是否可见（不等待） |

## 二、输入框操作

### 文本框（input / textarea）

| 操作 | 方法 | 说明 |
|------|------|------|
| ⌨️ 输入文本 | `.fill('文本')` | 单行 `input` 或多行 `textarea` 均适用 |
| 🗑️ 清空内容 | `.clear()` | 清空输入框 |
| 📖 获取输入值 | `.input_value()` | 获取用户输入的文本内容 |

### 文件输入框

```python
# 设置选中的文件
page.locator('input[type=file]').set_input_files('file.txt')
```

### Radio 单选按钮

| 操作 | 方法 |
|------|------|
| ✅ 选中 | `.check()` |
| ❌ 取消选中 | `.uncheck()` |
| 🔍 判断是否选中 | `.is_checked()` |

### Checkbox 多选框

| 操作 | 方法 |
|------|------|
| ✅ 勾选 | `.check()` |
| ❌ 取消勾选 | `.uncheck()` |
| 🔍 判断是否勾选 | `.is_checked()` |

### Select 下拉菜单

```python
# 使用 select 元素对应的 Locator 对象的 select_option 方法
page.locator('select').select_option('option_value')
```

## 三、网页操作

| 操作 | 方法 | 说明 |
|------|------|------|
| 🌐 打开网址 | `page.goto('url')` | 导航到指定页面 |
| 🔄 刷新 | `page.reload()` | 刷新当前页面 |
| ⬅️ 后退 | `page.go_back()` | 浏览器后退 |
| ➡️ 前进 | `page.go_forward()` | 浏览器前进 |
| 📄 获取页面源码 | `page.content()` | 获取整个网页的 HTML |
| 📌 获取标题 | `page.title()` | 获取网页标题 |
| 📐 设置视口 | `page.set_viewport_size({"width": 1920, "height": 1080})` | 设置浏览器窗口尺寸 |
