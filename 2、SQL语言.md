# 📝 SQL 语言

SQL 是**数据库标准化语言**，所有关系型数据库都适用。

---

## 一、SQL 语法规则

| 规则 | 说明 |
|------|------|
| 大小写 | SQL 语句不区分大小写，**关键字建议大写** |
| 书写格式 | 可以单行或多行书写，以**分号**结尾 |

---

## 二、SQL 分类

### DQL — 数据查询语言

> 用于从表中查询数据

| 关键字 | 说明 |
|--------|------|
| `SELECT` | 查询数据 |
| `FROM` | 指定数据来源 |
| `WHERE` | 条件筛选 |
| `ORDER BY` | 排序 |
| `HAVING` | 分组后筛选 |

### DML — 数据操作语言

> 针对**表中的数据**进行操作

| 关键字 | 说明 |
|--------|------|
| `INSERT` | 添加数据 |
| `UPDATE` | 更新数据 |
| `DELETE` | 删除数据 |

### DDL — 数据定义语言

> 针对数据库对象（数据库、表、索引、视图等）

| 关键字 | 说明 |
|--------|------|
| `CREATE` | 创建数据库对象 |
| `ALTER` | 修改数据库对象 |
| `DROP` | 删除数据库对象 |
